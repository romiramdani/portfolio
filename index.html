My Story

Hallo, nama saya Romi Ramdani, lahir pada 15 Oktober 2004. Alamat di desa Jatimulya kec. Cidahu, kab. Kuningan, Jawa Barat. Saat ini saya kuliah di Universitas Pamulang program studi teknik informatika. Bahasa pemrograman yang saya kuasai diantaranya Javascript untuk di bagian Frontend bersama dengan HTML dan CSS, serta di bagian Backend saya menguasai Node js untuk pembuatan API

Saya mulai mengenal pemrograman pada awal tahun 2022 pada saat saya duduk di kelas 12. Semasa SMK saya tidak berada dijurusan IT, oleh karena itu, semua materi pemrograman tidak bisa saya dapatkan dari pembelajaran formal di sekolah, namun hal tersebut tidak menyurutkan semangat saya untuk belajar pemrograman khususnya di pemrograman web, bahkan itu menjadikan motivasi tersendiri bagi saya pribadi.
